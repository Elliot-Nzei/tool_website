## ✅ PAGE 1: `index.html` — Dashboard

### 🔧 Purpose

A centralized dashboard showing:

* Quick stats across Resume, Finance, Health, and Image modules
* Action shortcuts
* Project tracker section (laptop view only)
* Responsive UI with **mobile tabbar** and **laptop-only navbar**

### 📱 Mobile Layout

#### ✅ UI Blocks

* **Navbar**

  * Left: `ToolMaster` logo or app name
  * Right:

    * 🔔 Notification icon
    * ➕ Module launcher
    * 🌙 Dark mode toggle

* **Main Content**

  * Grid of **4 stat cards** (Resume, Finance, Health, Image)
  * 4 large **action buttons**
  * (Laptop only) project status section

* **Tabbar (Fixed bottom)**

  * 🏠 Home (active)
  * ➕ (module selector)
